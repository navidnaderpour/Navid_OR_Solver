\# Navid OR Solver



A Python-based graphical application for solving Operations Research problems.



\---



\## 🔧 Features



\### 1. Simplex Solver

\- Standard Simplex

\- Big M Method

\- Two-Phase Method



\### 2. Transportation Solver

\- Initial solution methods

\- Optimization of transportation cost



\### 3. Network Solver

\- Shortest Path (Dijkstra)

\- Maximum Flow (Ford-Fulkerson)

\- Minimum Spanning Tree (Prim / Kruskal)
